NodeJS ACR122U Access Control System 
 
Installation Instructions: 
1. Extract all files to C:\Program Files\NodeJS_ACR122U\ 
2. Open Command Prompt as Administrator 
3. Navigate to the installation directory 
4. Run: node install-service.js 
5. The service will start automatically 
6. Open http://localhost:3007 in your browser 
 
To uninstall service: 
Run: node uninstall-service.js
